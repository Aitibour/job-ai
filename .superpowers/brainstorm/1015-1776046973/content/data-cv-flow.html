🤖 CV Tailoring Flow — what happens when you click "Tailor CV"
1. You click "Tailor CV"
On any job in the dashboard. The job description is already in Supabase.
↓
→
2. Claude API called
Receives your base CV + job description. Prompt instructs: match keywords, reorder bullet points, highlight relevant experience, keep it honest.
↓
→
3. ATS score calculated
Claude also returns matched keywords and an ATS alignment score (0–100). Both saved to cv_versions.
↓
→
4. PDF generated
Tailored CV rendered to PDF via react-pdf. Stored in Supabase Storage. You see a preview + download link.
↓
→
5. You apply
Review CV draft, then click "Apply Now →" which opens the job URL. Mark as Applied — date + company logged instantly.
🏨 Hospitality-specific CV tailoring
When the job is tagged Hospitality, Claude's prompt explicitly leads with your Oracle OPERA Cloud migrations, Marriott / Barcelo / Mandarin Oriental experience, and zero-downtime go-lives. For non-hospitality roles, it leads with your PMP, PMO governance, and infrastructure credentials instead.
